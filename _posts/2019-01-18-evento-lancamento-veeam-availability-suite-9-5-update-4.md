Como Veeam Vanguard, tenho acompanhado de perto a evolução desse produto ao longo do tempo. A atualização no momento, irá englobar os produtos core da Veeam: <strong>Veeam Backup &amp; Replication</strong>, <strong>Veeam ONE</strong>, V<strong>eeam Agent <em>for Microsoft Windows</em></strong> e <strong>Veeam Agent <em>for Linux</em></strong>.

Algumas das novidades que serão apresentadas no evento do dia 22/01 são:

<strong>Expansão do gerenciamento de dados na Cloud (AWS, Azure e IBM Cloud)</strong>
<ul>
 	<li>Retenção de dados na nuvem de uma forma simplificada, permitindo os dados "long term" serem armazenados utilizando object storage de uma forma nativa</li>
 	<li>Cloud Mobility - Facilidade na portabilidade e recuperação de dados entre seu ambiente on-premises e a nuvem</li>
 	<li>Segurança e Compliance - Melhoria nas políticas de segurança e compliance com o GDPR.</li>
</ul>
<strong>Novidades do Veeam Availability Suite 9.5 Update 4</strong>
<ul>
 	<li>Veeam Cloud Tier</li>
 	<li>Veeam Cloud Mobility</li>
 	<li>Veeam DataLabs</li>
 	<li>Veeam Intelligent Diagnostics</li>
</ul>
